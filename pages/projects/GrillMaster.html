Grill Master

Identidad visual y desarrollo web para Grill Master

Descripción del Proyecto

GrillMaster es una empresa ficticia dedicada a ofrecer servicios de parrilladas para eventos, proporcionando una experiencia gastronómica única con carnes de alta calidad y un servicio profesional.

El proyecto incluye la creación de una identidad visual sólida y el desarrollo de una plataforma web donde los clientes pueden reservar eventos, explorar menús y contactar con el equipo.
